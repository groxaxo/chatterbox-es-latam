# Web UI - Chatterbox ES-LATAM

Interfaz web moderna para el servidor TTS Chatterbox ES-LATAM.

## Tecnologías

- React 19
- Vite 6
- CSS moderno

## Desarrollo

```bash
cd web
npm install
npm run dev
```

## Build

```bash
npm run build
```

Los archivos compilados se generan en `dist/` y se sirven desde el servidor FastAPI principal.

## Uso

La UI está disponible en `http://localhost:8004` cuando el servidor está corriendo.
