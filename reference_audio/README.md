# Audio de Referencia

Este directorio almacena los audios de referencia subidos por los usuarios para clonación de voz.

## Clonación de Voz

El sistema permite clonar voces usando muestras de audio de referencia. Los archivos subidos se almacenan aquí temporalmente.

## Requisitos del Audio

Para obtener los mejores resultados en la clonación:

- **Formato**: WAV, MP3, o FLAC
- **Duración**: 3-10 segundos (óptimo)
- **Calidad**: Audio claro y nítido
- **Contenido**: Voz hablando naturalmente
- **Sin ruido**: Minimizar ruido de fondo
- **Mono o Estéreo**: Ambos funcionan

## Uso

Coloca manualmente los archivos de audio de referencia en este directorio para usarlos con el modo de clonación de voz a través de la API.

## Limpieza

Los archivos en este directorio pueden eliminarse periódicamente para liberar espacio.
El servidor no los elimina automáticamente.

## Privacidad

⚠️ **Importante**: Los archivos de audio pueden contener información sensible.
Asegúrate de implementar las medidas de privacidad apropiadas si despliegas este servidor en producción.
